# PowerVote

improve origin vote in weiphp v2.0

# Feature

* use bootstrap
* optional require wecaht subscribe
* WYSIWYG Editer
* Beautify UI

# Deploy

You just only to install `fabric` and run 
```
fab -H HOST deploy:PATH
```

Replace HOST and PATH with you need
